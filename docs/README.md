- MC : "경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)"

- MC : "시도할 회수는 몇회인가요?"
- player : n의 값 받기(자동차는 전진 또는 멈춤) (0~9)

- 실행 결과
- "최종 우승자 :" + 자동차 이름

### - 구현 목표
- MC가 말하는 것 출력
- 자동차 클래스 생성 
- 자동차 이름 값이 조건에 맞지 않을 경우 IllegalArgumentException 발생
- 사용자의 이동 횟수 받고 해당 횟수만큼 진행
- 0~9 사이 무작위 값 후 4이상일 경우 전진
- 우승자 출력
